Command Examples
File Encryption

Encrypt a file:

python encrypt.py --encrypt --file document.pdf

Encrypt with custom output:

python encrypt.py --encrypt --file document.pdf --output secure_doc.bin
File Decryption

Decrypt a file:

python encrypt.py --decrypt --file encrypted.bin --key abc123def456

Decrypt with auto-naming (creates decrypted_filename.ext):

python encrypt.py --decrypt --file encrypted_document.pdf.bin --key abc123def456

Security Guidelines
Important Security Notes:
  • Key Storage: Always save your Key IDs securely - they cannot be recovered
  • File Cleanup: Original files are automatically removed after encryption
  • Backup: Keep backups of important files before encryption
  • Key Sharing: Never share Key IDs through insecure channels
  • File Naming: Decrypted files are automatically named with "decrypted_" prefix
  • File Organization: All encrypted and decrypted files are stored in "encrypted_decrypted_files" directory
